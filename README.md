# Belle & Bloom Analytics - Assignment 5 Dashboard

This is a high-fidelity bilingual interactive dashboard prototype for SEG3125 Assignment 5.

## Project theme

The dashboard extends the Belle & Bloom e-commerce site from Assignment 4. It uses synthetic data to show sales and customer insights for clothing and beauty products.

## Main features

- React high-fidelity prototype
- Two chart types:
  - Line chart for monthly revenue trend
  - Bar chart for product performance
- Interactions:
  - Language selection: English / French
  - Currency selection: CAD / EUR
  - Category filters
  - Metric selection for the bar chart
  - Reset dashboard button
- Localization:
  - Interface text
  - Chart labels
  - Month names
  - Currency and number formatting
- Accessibility:
  - Keyboard-focus styles
  - Skip link
  - SVG titles/descriptions
  - Accessible data tables below charts
